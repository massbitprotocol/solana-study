# solana

# Run validator
```
solana-test-validator
```
# Check config
```
solana config get
```
# Check Balance
```
solana balance
```
# Create new account
```
solana-keygen new
```
# Start new validator with new account
```
solana-test-validator -r
```
# Deloy program on solana
```
solana deploy PATH_TO_YOUR_PROGRAM
```
